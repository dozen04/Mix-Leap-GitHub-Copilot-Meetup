# 第3回 プロダクト戦略会議 サマリー

**日付**: 2026-03-02
**参加者**: 山田（PM）、中村（エンジニアリーダー）、高橋（デザイナー）、佐々木（QA）、田中（マーケティング）

## 会議のサマリー

Q2ロードマップの進捗確認と、新機能「ダッシュボード v2」のスコープ、ユーザーフィードバック対応、QA体制の見直しについて方針決定を行った。遅延リスクと品質体制を踏まえ、段階的リリースと自動化強化を軸に進める方向で整理した。

- Q2主要マイルストーン3件中2件は順調、通知機能リニューアルは設計変更で遅延中
- ダッシュボード v2 は案A（カード型）を採用し、2フェーズで段階的に実装する方針を確認
- NPS低下要因（検索速度・モバイル操作性）に対し、検索改善を先行対応する方針を確認
- QA体制不足への対策として、Playwright導入によるE2E自動化強化を承認

## 合意事項

- ダッシュボード v2 は**案A（カード型レイアウト）**で進めることに決定した
- ダッシュボード v2 は**第1フェーズで基本レイアウト、第2フェーズでカスタマイズ機能追加**の2段階で進めることに決定した
- ダッシュボード v2 のQAスケジュールは**1週間追加**して調整することに決定した
- 検索機能改善は優先度を上げ、**次スプリントでインデックス最適化対応**することで合意した
- モバイル操作性の課題は、**ダッシュボード v2 のレスポンシブ対応で改善を図る**ことで合意した
- QA体制見直しとして、**Playwright導入を承認**し導入計画を作成することに決定した

⚠ 未決定

- 通知機能リニューアルの遅延が2週間を超えた場合に実施する具体的なスコープカット内容は未決定
- Playwright導入計画の詳細スケジュール・担当工数配分は未決定
